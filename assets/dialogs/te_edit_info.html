Use this page to edit a tracker you've previously created. A tracker contains the logs for a single event (e.g. getting a haircut, exercising, calling Mom).

You can save your changes when you're ready by hitting the "Save" button or the back button. Use the "Cancel" button to cancel your changes.

To delete this tracker, push the menu button and choose "Delete." You can also delete a tracker from the main screen (long press the entry and choose "Delete").
