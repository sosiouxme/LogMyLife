Use this page to create a tracker which will contain the logs for a single event (e.g. getting a haircut, exercising, calling Mom).

A tracker only needs a name in order to be created (the "Save" button is inactive until you specify one). You may also choose to add:

  • Notes clarifying what the tracker is for
  • A label for values that you record with log entries
  • Alerts to remind you when time has passed since logging an entry

You can save the tracker when you're ready by hitting the "Save" button or the back button. Use the "Cancel" button to cancel tracker creation.
