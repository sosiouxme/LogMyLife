LogMyLife is an application to help you track recurring events and (optionally) remind you when too much time has passed since the last occurrence.

Use the menu button on most screens for extra options. Also, most lists have extra long-click options.

This is an alpha release! Expect broken stuff! And please create issues for problem reports and requests (see below).

Usage: the main screen

This main screen shows a list of trackers you can define in order to track events. Once you have created a tracker, you can log entries to it.

Trackers may be organized into groups, e.g. "Household", "Personal", etc. The group is displayed and chosen at the top of this screen. You may create and edit groups by pressing the Menu button and selecting "Groups".

Liberate your data

All of the data you enter in this application can be exported to the SD card in order to be imported into another LogMyLife installation. To do this, press the Menu button and select "Data".

Future versions of LogMyLife should also be able to synchronize data to a website, as well as providing it in spreadsheet-readable format.
