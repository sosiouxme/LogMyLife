Use these buttons to create log entries for the tracker, which will show up as entries in the list below these details.

  • Log now: Add an entry for the current time, with no comments or value entered.
  • Log details: Add an entry with the opportunity to adjust time and fill in comments and a value.

Tip:These options are also available from the menu (press the Menu button) or from the main list of trackers by long-pressing on the tracker to get a context menu.

You can always edit a log's details later (just select it from the list).
